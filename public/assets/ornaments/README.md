# Ornaments

Decorative SVGs are split into two buckets:

## Raw file assets in `public/assets/ornaments/`

Use raw SVG files for large static art and CSS/background references:

- `dividers/`
- `corners/`
- `arches/`
- `lanterns/`
- `compass/`
- `routes/`
- `tape/`

## React component counterparts

Use `src/components/ui/decor/` for reusable, prop-driven ornament components:

- `SectionDivider`
- `CornerOrnament`
- `RouteLine`
- `CompassMotif`
- `ArchFrame`
- `LanternIcon`
- `ScrapbookTape`
